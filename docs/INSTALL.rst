wccpilgrimage.policy Installation
---------------------------------

To install wccpilgrimage.policy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wccpilgrimage.policy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wccpilgrimage.policy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
